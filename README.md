☕ Călătoria Cafelei în România
Proiect Laborator POO

Această aplicație permite gestionarea eficientă a unei rețele de cafenele din România, utilizând principii de Programare Orientată pe Obiecte (POO). Aplicația facilitează administrarea personalului, monitorizarea stocurilor, procesarea comenzilor și organizarea evenimentelor speciale.

🎯 Functionalități
📌 Gestionarea personalului – Administrarea angajaților fiecărei cafenele (barista, manageri, ospătari).
📦 Monitorizarea stocurilor – Evidența produselor disponibile (cafea, ceai, dulciuri, sandwichuri etc.).
🛒 Procesarea comenzilor – Înregistrarea și calcularea automată a costurilor fiecărei comenzi.
📅 Planificarea evenimentelor – Organizarea campaniilor promoționale și a evenimentelor speciale.
